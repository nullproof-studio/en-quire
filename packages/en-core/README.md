# @nullproof-studio/en-core

Reliability primitives shared by the [en-quire](https://github.com/nullproof-studio/en-quire/tree/main/packages/en-quire) and [en-scribe](https://github.com/nullproof-studio/en-quire/tree/main/packages/en-scribe) MCP servers: etag optimistic locking, proposal-gated writes, unified-diff generation, file/encoding utilities, git integration, RBAC, and the parser registry both siblings extend.

This package is a library. The executables live in en-quire and en-scribe.

See the [repo README](https://github.com/nullproof-studio/en-quire#readme) for positioning and usage.
